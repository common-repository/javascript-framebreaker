== Description ==

Adds a framebreaker JavaScript function to the header for breaking out of Google Image Search and other framesets in order to keep users on the site. Mainly for [SEO](https://www.search-one.de/).

== Installation ==

1. Upload `wp_framebreaker.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
